# Code Example

```python
# Sample Python code for demonstration

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```

---

# Algorithmic Complexity

For the recursive Fibonacci function:

\[
T(n) = T(n-1) + T(n-2) + O(1)
\]

Time complexity:

\[
O(2^n)
\]
